# 项目简介

## 核心功能
- 用户管理模块
  - 用户信息管理：允许用户修改个人信息和密码。
- 问卷管理模块
  - 问卷创建：支持创建多种题型问卷，设置问卷基本信息和逻辑跳转。
  - 问卷发布：设定问卷填写次数限制。
  - 问卷编辑与删除：编辑和删除已创建问卷。
  - 问卷管理：展示所有问卷，支持搜索、筛选和排序。
- 问卷填写模块
  - 问卷填写：动态加载题目，支持多种题型展示。
  - 数据校验：对用户填写数据进行校验。

## 适用场景
- 教育机构进行问卷调查。
- 市场研究部门收集用户反馈。
- 企业进行员工满意度调查。
- 社会组织进行公众意见征集。

## 技术特点
- 使用Python进行开发。
- 支持多种题型，满足不同调查需求。
- 界面友好，操作便捷。
- 数据校验确保问卷数据准确。
